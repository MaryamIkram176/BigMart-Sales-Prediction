This project predicts the sales of Big Mart outlets using machine learning.  
The dataset is taken from Kaggle.

Steps:
- Data cleaning  
- Label Encoding  
- Train/Test split  
- Linear Regression model  
- RMSE calculation  
- Create submission.csv
  
Files:
- bigmart_sales.ipynb  
- train.csv  
- test.csv  
- submission.csv  
-sample_submission.csv
